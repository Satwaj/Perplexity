🚀 **Excited to share my latest project: Arena AI — a real-time LLM Benchmarking & Multi-Agent Chat Hub!** 🤖✨

I wanted to build a platform that goes beyond standard model chats by allowing users to run side-by-side comparative evaluations of state-of-the-art LLMs, complete with automated judging.

I built this with a modern **Neobrutalist design aesthetic** and highly interactive motion physics. Here’s a breakdown of the technical highlights:

💡 **Key Features & Architecture:**
*   **Dual LLM Battle Arena:** Run concurrent, asynchronous queries to multiple model endpoints (Mistral & Groq / LLaMA3) to render side-by-side solutions. An automated judge evaluates the answers, calculates points, and signs a final verdict.
*   **Real-time Streaming Pipeline:** Integrated WebSockets (Socket.io) with Redux Toolkit to maintain seamless, low-latency message streaming and state synchronization.
*   **Immersive Motion UI (GSAP):** Implemented continuous geometric background drift and a mouse-movement tracking parallax effect, along with staggered slide-in transitions on load.
*   **Web Speech Synthesis:** Created an interactive voice reader utilizing the Web Speech API for playback of assistant responses.
*   **Production Deployment:** Configured CORS and secure JWT authentication to whitelist clients across Vercel (Frontend) and Railway (Backend).

🛠️ **Tech Stack:**
*   **Frontend:** React.js, Redux Toolkit, GSAP, Tailwind CSS, Vite
*   **Backend & DB:** Node.js, Express, Socket.io, MongoDB, Mongoose, JWT
*   **AI Integration:** Groq API, Mistral AI, Gemini API
